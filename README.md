# AgentOCR
## **简介**
* AgentOCR 是一个基于 PaddleOCR 和 ONNXRuntime 项目开发的跨平台、易部署的 OCR 项目

* 支持 PaddleOCR 的多语言预训练模型，以及 PaddleOCR 训练导出的其他模型
